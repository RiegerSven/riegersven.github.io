Overview Complier Average Causal Effect (CACE)

Terminology

  • Outcome: \(Y_i\)

  • Assignment of treatment (and instrument): \(Z_i\)

  • Receipt of treatment (compliance status): \(W_i\)

Noncompliance

By noncompliance we refer to the situation where some units who are assigned to receive a particular treatment level do not comply with their assignment and instead receive an alternative treatment (imbens2015?)

One- and two-sided Noncompliance

One-sided noncompliance

One-sided noncompliance means that it is asymmetric. Only units that are assigned to receive a treatment can circumvent their assigned treatment. Units in the control condition comply with this assignment (imbens2015?).

Observed outcome: \(Y^{obs}_i = Y_i(Z_i,W^{obs}_i)= Y_i(Z_i,W_i(Z_i)) = \begin{cases} Y_i(0,0), \text{ if } Z_i = 0, W^{obs}_i = 0\\ Y_i(1,0), \text{ if } Z_i = 1, W^{obs}_i = 0 \\ Y_i(1,1), \text{ if } Z_i = 1, W^{obs}_i = 1 \\ \end{cases}\)

Example: Data generation

genOSNC <- function( n = 1000,
                     tau = .5,
                     bu = 0, du = 0,
                     compliance = "one-sided" ) {
    
    # treatment
    Z <- rbinom(n, size = 1, prob = .5)
    n1 <- sum(Z)
    # unobserved confounder
    U <- rnorm(n, 0, 1)
    
    dat <- as.data.frame(cbind(Z,U))
    
   
    # compliance    
        
    if ( compliance == "one-sided") {

      dat[dat$Z == 1, "W"] <- rbinom(n1,
                                     size = 1,
                                     prob = 1 / (1 + exp(- du*U)))  # check prob
      dat[dat$Z == 0, "W"] <- 0                  
      
    } else if (compliance == "two-sided") {

    } else {
      stop("state either one-sided or two-sided compliance status")
    }
    

    # outcome
    dat$Y <- with(dat, 0 + 0*Z + tau*W + bu*U + rnorm(n, 0 ,1))

    return(dat)

}

Estimating the ITT Effect for the Receipt of Treatment

(needed for LATE effect)

Complete randomization of the assignment implies that an unbaised estimator for the avarage causal effect \(ITT_W\) exists in the form of the average difference in treatment status by assigment status (imbens2015?, p, 520)

\(\widehat{ITT_W}=\overline{W}^{obs}_1-\overline{W}^{obs}_0 = \overline{W}^{obs}_1\)

ITTw <- mean(exDat[exDat$Z == 1, "W"])
